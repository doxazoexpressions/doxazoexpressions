## 5. Icon / splash confirmation

Already in repo:
- `public/icon-192.png`, `public/icon-512.png`, `public/apple-touch-icon.png`
- `public/manifest.webmanifest`
- Splash & status bar: dark navy `#0f1a2b`, configured in `capacitor.config.ts`

For native builds you'll regenerate the full asset set from 1024² source art:
```bash
npx @capacitor/assets generate --iconBackgroundColor '#0f1a2b' --splashBackgroundColor '#0f1a2b'
```

Source files to place locally before that command:
- `resources/icon.png` — 1024×1024, no transparency for App Store
- `resources/splash.png` — 2732×2732, gold "D" mark centered on navy

## 6. Pre-submission must-fix (small, code-side)

1. **Account deletion path** — Apple requires it because we support account creation. Add a "Delete account" button in `/settings` that calls an edge function to delete the user + their `favorites`, `push_subscriptions`, `device_tokens` rows. (Server-side cascade already in place for most tables; needs the explicit UI + edge function.)
2. **Remove `server.url` from `capacitor.config.ts`** before any release build, then `npx cap sync`. Live-reload URL must NOT ship to the store.
3. **iOS 26 SDK** — must build with Xcode 26+ per Apple's April 28, 2026 requirement. Verify your local Xcode version before archiving.
